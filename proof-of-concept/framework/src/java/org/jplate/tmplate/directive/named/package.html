This package provides useful implementations of org.jplate.tmplate.directive.NamedDirectiveIfc. It is important to note that when a directive container requests a named directive to translate, the named directive has the "ultimate say" as to what that translation becomes. One can examine the contents of the directive container and operate upon the contents or it can do something completely different. Some good examples are:
  • org.jplate.tmplate.directive.named.AliasDirective: when translating it names a directive (ie creates a new named directive) whose contents are those defined within the alias directive block. Essentially it takes the contents of its' directive container, creates a new named directive who will return those contents when asked to translate.
  • org.jplate.tmplate.directive.named.DateDirective: when translating it simply computes the date/time based upon a filtering string.
