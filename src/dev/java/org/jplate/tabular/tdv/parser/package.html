Contains definitions for TDV (tabbed delimited values) parsers.

An example of TDV data follows:

  • field 1-1\tfield 1-2\tfield 1-3
  • field 2-1\tfield 2-2\tfield 2-3
  • field 3-1\tfield 3-2\tfield 3-3
